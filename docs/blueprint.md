# **App Name**: SpendWise

## Core Features:

- Add Expense: Allow users to add new expenses by specifying amount, category, and description.
- Expense List: Display expenses in a clear, chronological list.
- Category Management: Allow users to create and manage spending categories.
- Spending Summary: Show a summary of total spending, broken down by category and time period (e.g., daily, weekly, monthly).

## Style Guidelines:

- Primary color: Soft green (#90EE90) to convey a sense of financial well-being.
- Background color: Very light green (#F0FFF0), nearly white, for a clean and airy feel.
- Accent color: A slightly darker green (#3CB371), will be used for interactive elements and highlights to guide user attention and signal actions.
- Body and headline font: 'PT Sans', a modern humanist sans-serif for readability.
- Use simple, minimalist icons to represent expense categories for easy identification.
- Maintain a clean, uncluttered layout to ensure a user-friendly experience.
- Incorporate subtle animations (e.g., when adding a new expense or transitioning between views) to enhance usability.
